# LA Board Game

This is a browser-based board game made with HTML, JavaScript and CSS for an LA project. It adapts themes from the novel "The Slave Dancer."

## How to play

This will provide a basic overview of how to play, however there is a ```RULES.md``` for a deeper dive. 

There must be at least 2 players and a maximum of 4 players.

Each turn:
- The current player rolls a standard 6-sided die
- Move forward that many spaces along the track (the board uses 28 spaces)
- Read the event for the space and choose one of the presented options

Endgame & winning:
- When a player reaches the final space (space 28) the game enters endgame mode: every other player receives one final turn.
- Final scoring uses the sum of the three core stats (Survival + Humanity + Resolve). The player who first reached the final space receives a +2 shore bonus when scores are compared.

## Installation

To run the game locally, download or clone the repository (including the `resources/` folder) and open `index.html` in a modern browser (Chrome, Edge, Firefox, etc.). No build step is required.

**Requirements:** Modern browser with JavaScript enabled. No additional dependencies.

Note: The `resources/` folder contains image and media assets used by the game (board background, player video token, etc.).
